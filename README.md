## 🌟 About

Welcome to **OZ Chicken Burger** - a modern, responsive web application showcasing one of Los Angeles' finest burger experiences. This website brings the sizzle of our kitchen right to your screen, featuring our mouthwatering menu, captivating gallery, and everything you need to know about our restaurant.

Built with cutting-edge web technologies, this project delivers a seamless and delicious browsing experience across all devices.

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shaflink23/Oz_chicken_burger.git
   cd Oz_chicken_burger
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:5173
   ```
